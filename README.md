# Chippure
A Chip-8 emulator written using Python's default libraries.
###Instructions:
Simply launch the .py file and type the name of the Chip8 ROM you want to launch, then push enter.
**Keyboard layout:**
  1 2 3 4
  Q W E R
  A S D F
  Z X C V
